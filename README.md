# AstrBot WOL 简易局域网唤醒插件

局域网唤醒插件，支持绑定设备、发送唤醒包、在线状态检测。

## 指令说明
- `/绑定 <MAC地址> [IP地址]` - 绑定设备
- `/开机` - 发送唤醒包，30秒后检查状态
- `/我的电脑` - 立即查询设备状态
- `/局域网唤醒帮助` - 显示帮助

## 配置项
- `mac`: 绑定的 MAC 地址
- `ip`: 绑定的 IP 地址
- `broadcast`: 广播地址（默认 255.255.255.255）
- `port`: 端口（默认 9）
- `allowed_users`: 允许使用的用户 ID 列表

## 安装
1. 将插件放入 `data/plugins/` 目录
2. 重启 AstrBot 或执行 `reload`
3. 确保已安装依赖：`pip install ping3`
