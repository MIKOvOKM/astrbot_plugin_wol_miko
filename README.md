# AstrBot WOL 简易局域网唤醒插件

局域网唤醒插件，支持绑定设备、发送唤醒包、在线状态检测。
## docker模式注意
1. docker的桥接模式下广播可能失效，推荐使用单播，也就是配置固定的电脑IP。需要宿主机能访问此IP
2. 若需在docker模式下使用，请切换为host模式且宿主设备和电脑在同一局域网。

## 指令说明
- `/绑定 <MAC地址> [IP地址]` - 绑定设备
- `/开机` - 发送唤醒包，30秒后检查状态
- `/我的电脑` - 立即查询设备状态
- `/局域网唤醒帮助` - 显示帮助

## 配置项
- `mac`: 绑定的 MAC 地址
- `ip`: 绑定的 IP 地址 #未配置时，无法准确查看在线状态，无法使用单播
- `broadcast`: 广播地址（默认 255.255.255.255）#广播配置，使用单播可无视
- `port`: 端口（默认 9）
- `allowed_users`: 允许使用的用户 ID 列表-默认仅限私聊 #未配置时，无法正常使用功能。

## 安装
1. 将插件放入 `data/plugins/` 目录
2. 重启 AstrBot 或执行 `reload`
3. 确保已安装依赖：`pip install ping3` #启动时自动安装，若未安装需手动
